# Claude Code Harness Appendix

## Capability routes

Load shared skills only when they materially improve the result.

Common routes:

- `skill-discovery` when the capability route is unclear
- `elegy-planning` for durable planning state
- `rubberduck-plan-review` before complex plan execution
- `spec-dev` for an end-to-end spec workflow
- `skill-authoring` for shared skill authoring
- `agents-md-authoring` for instruction layering and instruction-file audits
